Muy Importante

El proyecto está desarrollado en JDK 21, por lo tanto se recomienda configurar el entorno con esta versión para evitar problemas de compatibilidad.

El entorno de desarrollo utilizado fue IntelliJ IDEA.

En el repositorio se incluirán capturas de pantalla desde IntelliJ mostrando la configuración del SDK y la ejecución del proyecto.

Como ejecutar el proyecto

Clonar el repositorio.

Abrir el proyecto en IntelliJ.

Verificar que el Project SDK esté configurado en Java 21.
<img width="1090" height="965" alt="image" src="https://github.com/user-attachments/assets/26b4ea89-364b-4ab7-a0d9-bf5beda28e92" />


Ejecutar la aplicación Spring Boot.

Una vez iniciada correctamente, acceder desde el navegador a:

http://localhost:8080/index.html


Desde esa URL se puede utilizar la interfaz gráfica para probar los algoritmos de ordenamiento y búsqueda.
